Hướng dẫn cài đặt angular

Để cài đặt angular, đầu tiên ta new 1 cái angular project

ng new frontend

Sau khi khởi tạo nó rồi, đơn giản là ta sẽ start nó thôi đúng ko

ng serve --open

--open khai báo tự open cái browser lên để show cái page.

Các npm cần install để làm việc trong project này bao gồm:

  1. material
  2. boostrap
  3. node sass (để generate scss files)

Sau đó, ta add material cho cái project này

ng add @angular/material

Add luon cai boostrap

npm i -S bootstrap.

Xem hướng dẫn install bootstrap cho angular tai : đây

Install node sass để generate scss file. Nếu ko cài cái này thì style trong file scss sẽ ko được generate ra đâu

npm install -g node-sass

Hướng dẫn tạo component và khai báo routers

Để tạo một component tự động, ta làm như sau : ** Khuyến khích xài command line vì generate thủ công rất dễ sai và đôi khi xảy ra những sai sót ko đáng có.

  1. cd vào folder cài đặt angular, folder chứa file package.json (ở đây là front-end)
  2. Dùng command line sau để tạo component một cách tự động:
    ng g c components/help

    Giải thích thêm : g - generate, c - component, components/help : tạo component Help ở folder /components

Tạo service trong Angular

Việc tạo service cũng như tạo component, tốt nhất nên generate tự động.

ng g s issue

Khác với component, sau khi generate xong ta phải khao báo trong declarations của app.module, thì service phải được khai báo trong providers.

Như vậy, trước mắt ta có các lưu ý sau:

  1. Component khai báo trong declarations
  2. Service khai báo trong providers
  3. Module khai báo trong imports
